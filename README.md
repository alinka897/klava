# Описанине проекта.

Данная программа считывает текстовый файл и выводит статистику по 7 раскладкам: ЙЦУКЕН, фонетическая, диктор, скоропись, ант, Зубачёв, вызов. Статистика включает в себя:
* Число штрафов на каждый палец каждой руки (т.н. нагрузка на пальцы);
* Распределение нагрузки в процентном соотношении для правой, левой и обеих рук;
* Удобные переборы для обеих рук;
* Переборы для правой и левой руки: как по числу штрафов, так и в процентном соотношении.

# Как использовать проект klava.
При запуске программа предложит ряд вопросов:
### Как считаем штрафы?
1. Статически
2. Динамически
### Что хотим сделать?
1. Проланализировать одну раскладку
2. Сравнить несколько раскладок
### Выберите раскладку(-и):
1. ЙЦУКЕН
2. Фонетическая
3. Диктор
4. Скоропись
5. ANT
6. Зубачёв
7. Вызов
При выборе сравнения нескольких раскладок введите номера нужных через пробел.
### Что хотим сделать?
1. Прогнать через файл
2. Штрафы построчно
### Результат:
После введения пути к текстовому файлу, выводится результат в виде двух типов диаграмм: линейной и круговой. При выборе "Штрафы построчно" в дополнение к диаграммам создаётся файл result.txt с индивидуальными результатами для каждой строчки текстового файла.
# Статические штрафы.
На линейной диаграмме представлено количество нажатий для каждого пальца обеих рук, а на круговой - процентное соотношение использования правой, левой и обеих рук одновременно при наборе выбранного текстового файла. # Динамические штрафы.
На двух линейных диаграммах представлены удобные переборы для правой и левой руки. Переборы сгруппированы по 2, 3, 4 и 5 символов. На круговой диаграмме показано процентное соотношение переборов вида: 
* У - удобные (т.е. переход пальцев от внешних к внутренним),
* ЧУ - частично удобные (т.е. смена направления пальцев на одной руке),
* НУ - неудобные (т.е. смена рук - сначала правая, потом левая и т.д.).

# Вывод.
Целью этой программы является выяснение какая раскладка является лучшей по критерию "путь пальца". Ниже будут представлены результаты раскладок при наборе ткстового файла prim.txt в формате:
```bash
название раскладки [самый большой путь, т.е. сумма штрафов всех пальцев] число удобных двусимвольных переборов (левая + правая рука) 
```
* ЙЦУКЕН       [2372]  163
* Фонетическая [2004]  183
* Диктор       [1381]  23
* Скоропись    [1174]  7
* ANT          [2180]  17
* Зубачёв      [1775]  27
* Вызов        [1971]  77

Наименьшее число штрафов собрала раскладка "скоропись". Наибольшее число удобных двусимвольных переборов (по отношению к числу штрафов) имеет "фонетическая" раскладка.
